##### 使用手机号注册账号

* 请求URL

> [http://118.31.2.202:8080/zhibo/user/registerByPhone](http://118.31.2.202/user/registerByPhone)

* request字段示例:

> {phone:"17858936666",password:"123456;username:"xxx""}

| 字段 | 类型 | 选项 | 说明 |
| :---: | :---: | :---: | :---: |
| phone | String | 必填 | 手机号 |
| password | String | 必填 | 密码 |
| username | String | 必填 | 用户昵称 |

* response字段示例

> {errorCode:"0",errorInfo:""}
